Mode de Fonctionnement Interne

Pour réaliser ce projet, chaque membre du groupe avait un rôle précis :

  • Jean Dupont : Chef de projet et développeur backend. Jean a coordonné les efforts de l'équipe et a développé l'API du serveur en utilisant Django.
  • Marie Curie : Développeuse frontend. Marie a conçu et codé l'interface utilisateur du site en utilisant Django templates et du JavaScript.
  • Albert Einstein : Architecte de la base de données. Albert a conçu la structure de la base de données en utilisant Django ORM et a implémenté les modèles de données.
  • Isaac Newton : Analyste en sécurité. Isaac a veillé à ce que le site soit sécurisé contre les attaques potentielles et a implémenté les bonnes pratiques de sécurité dans Django.
  • Rosalind Franklin : Spécialiste UX/UI. Rosalind a travaillé sur l'expérience utilisateur et l'ergonomie du site.

Description de la Conception

Structure du Projet

Le projet est structuré de la manière suivante :

  • Frontend : Utilisation de Django templates pour construire l'interface utilisateur. Du JavaScript et du CSS ont été utilisés pour améliorer l'interactivité et le style.
  • Backend : Utilisation de Django pour gérer les requêtes serveur, l'API REST avec Django REST framework.
  • Base de Données : Utilisation de PostgreSQL pour stocker les données des utilisateurs et des messages.
