Description de la Conception

Structure du projet

Le projet est structuré de la manière suivante:

  • Frontend: Utilisation de Django templates por construire l'interface utilisateur. Du JavaScript et du CSS ont été utilisés pour améliorer l'interactivité et le style.
  • Backend: Utilisation de Django pour gérer les requêtes serveur, l'API REST avec Django REST framework
  • Base de données: Utilisation de MySQL pour stocker les données des utilisateurs et des messages

Base de Données

La base de données est structurée en plusieurs tables:

  • Users: stocke les informations des utilisateurs (nom, prénom, email, mot de passe, etc.).
  • Profiles: stocke les informationsde profil des utilisateurs (age, sexe, intérets, etc.).
  • Messages: stocke les mesages échangés entre les utilisateurs
  • A améliorer et en ajouter

Instructions de Déploiement

Pour déployer le site de rencontres en ligne, suivez les étapes ci-dessous:

  • Clonez le dépot du projet depuis Github: git clone https://github.com/OlympBlack/PIL1_2324_11.git
  • Accédez au répertoire du projet: cd PIL1_2324_11
  • Créez et activez un environnement virtuel: python -m venv env && source env/bin/activate
  • Installez les dépendances: pip install -r requirements.txt
  • Configurez les variables d'environnement dans un fichier .env(incluant les informations de la base de données, db_name, db_password, user_name, db_host, db_port).
  • Appliquez les migrations de la base de données: python manage.py migrate
  • Créez un superutilisateur: python manage.py createsuperuser
  • Démarrez le serveur Django: python manage.py runserver
  • Accédez au site via votre navigateur à l'adresse: http://localhost:8000

Utilisez la barre de recherche pour trouvez des profils correspondant àà vos critères. Vous pouvez filtrer les résultats par âge, sexe, intérêts, etc.

Pour un message à un autre utilisateur, visitez son profil et cliquez sur "Envoyer un message". Vous pouvez consulter vos messages reçus dans la section "Messagerie".
